# VibeHacker88 Website

[![Deploy to GitHub Pages](https://github.com/vibehacker88/vibehacker88.github.io/actions/workflows/deploy.yml/badge.svg)](https://github.com/vibehacker88/vibehacker88.github.io/actions/workflows/deploy.yml)

A website about number puzzles and mathematics.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.
